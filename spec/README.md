# Спецификация сайта Eagle Security

Документы для подготовки и реализации сайта приложения Eagle Security.

## Файлы

| Файл | Описание |
|------|----------|
| [00-site-overview.md](00-site-overview.md) | Обзор: как сайт устроен, обязательные разделы (Impressum, Datenschutz), стиль, два языка, деплой. Конкретный набор страниц не задаётся. |
| [01-tz-template.md](01-tz-template.md) | **Шаблон ТЗ** — заполняется перед разработкой. Страницы, контент, дизайн, технология, скрипты. |
| [AI-INDEX.md](AI-INDEX.md) | Инструкция для AI: порядок чтения spec и правила реализации сайта по ТЗ. |

## Как пользоваться

1. Прочитайте [00-site-overview.md](00-site-overview.md).
2. Заполните [01-tz-template.md](01-tz-template.md) (можно частично; неподходящее — удалить или отметить «skip»).
3. Передайте заполненное ТЗ и ссылку на [AI-INDEX.md](AI-INDEX.md) разработчику или AI для реализации в папке `src/`.

Исходники и сборка сайта лежат в **`../src/`**.
